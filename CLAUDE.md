# 📱 プロジェクト: わたしはダレでしょう？クイズ集！

## 1. 開発チーム体制（自律連携モード）
Claude Codeは以下の専門家チームとして自律的に連携し、修正ごとに `git add / commit / push` を自動で行うこと。

**本番マージについて**: なつみより、開発ブランチでの修正が完了し動作確認(ビルド・
テスト)ができたら、都度「メインにマージして」と確認を取らずに、Claude Code の
判断で `main` へのPRマージ（本番デプロイ）を自律的に行ってよいと許可を得ている。
ただし、テストが通らない・重大な仕様変更・課金/決済まわりなど影響が大きい変更は、
念のためなつみに一声かけてからマージすること。
* **👑 PM**: 設計書(Ver. 1.8)の遵守と「なつみの願い」の管理。
* **🛠️ Lead Engineer**: タップ操作ロジック、Firebase、Stripe、バグゼロの徹底。
* **🎨 Creative Director**: 「脱・AI感」。子供向けの可愛らしく、ほんわかしたデザインへの統一。
* **💰 運営・法務**: 決済フロー完備、特定商取引法に基づく表記の管理。
* **📚 Researcher**: クイズの正確性（嘘の排除）、Lv1-10の難易度設計、JSONデータ検閲。

## 2. 絶対厳守：なつみの願い & 重要仕様
### 🚀 動作・ロジック (現状の最優先事項)
- **ハンズフリーモード**: **【実装不要】**。UI上は「近日公開」のラベルを表示。
- **タップ操作**: 通常モードでの4択クイズが完璧に動作すること。
- **ループ制**: 正解または降参が出るまで次問へ進まない。

### 🎨 デザイン・UI
- **コンセプト**: AIっぽくない、子供受けする可愛らしいデザイン。
- **デバイス対応**: スマホ表示を完璧にし、PCでもレイアウトが崩れないようにする。
- **アイコン**: 現在の乱れを修正し、統一感のある可愛いアイコン（商用フリー）を使用。

### 💳 課金・制限 (なつみの願い優先)

**⚠️ 現状(実装済みの事実)**: 問題数は各ジャンル×難易度ごとに固定のJSONデータ件数
(多くのジャンルは20問、魚類は50問、鳥類はレベルにより24〜50問)。購入は「その
レベルへのアクセス権を解放する」だけで、**購入によって問題数が動的に増える機能
は存在しない**。既読管理・Firestore同期・未回答優先機能も未実装。PaywallModal等
のUI文言は、必ず実際のデータ件数を動的に表示すること(固定文言で「◯◯問追加」
のように嘘の数字を書かない)。

**🎯 なつみの理想(未実装・将来実装したい仕様)**:
- 無料枠: 50問 → 「問題数UP」購入で追加50問(計100問)
- 有料枠: レベル解放で50問 → 「問題数UP」購入で追加50問(計100問)
- これは「レベル解放」とは別の、**2種類目の購入導線**として新設する必要がある
  (同じレベル内で最初は50問だけ遊べ、追加購入で残り50問が解放される、という
  出し分けロジックが必要)。実装には、全ジャンル・全レベルを最低100問まで拡充
  する作業も必要。着手する際はなつみに確認してから進めること。

- **購入前（ゲスト）**: 解放レベル: **1, 2, 6, 7**（こどもとおとな体験用）。
- **購入後 (120円)**: 対象レベルへのアクセスを解放。
- **まとめ買い割引 (2026年7月なつみ承認済み、実装予定)**:
  - ジャンル内まとめ買い(全13ジャンル共通、有料レベル6つ): 通常720円相当 → **330円**
    (歴史上の人物も2026年7月にLv1〜10・各50問へ拡充済みのため、他ジャンルと同額330円に統一)
  - 全ジャンル・全レベル解放: 通常9,360円相当 → **1,480円**
- **法的表記**: 取引に必要な情報（特商法等）を不足なくページとして用意すること。
- **ジャンルごとのレベル制限**: 以前は「歴史上の人物」が[6, 7, 8, 9, 10]のみ、「道具」が
  [2, 6, 9]のみに絞られていたが、なつみの指示によりいずれもLv1〜10全レベル×各50問に
  拡充済み(2026年7月)。現在は全13ジャンルすべてLv1〜10の全レベルが選択可能で、ジャンル
  ごとのレベル制限は存在しない。

### 📧 お問い合わせ・運営
- お問い合わせは、なつみの個人メールアドレス宛にメーラーが起動する（mailto）機能を実装。
- クイズデータは最新の生物学・歴史データに基づき、嘘偽りがないか徹底検証すること（Lv1-10まで）。
